Rewind expects a JSON file* in the following format:
{ "locations" : [ { "timestampMs" : "1467228482403", "latitudeE7" : 418261352, "longitudeE7" : -714042911, "accuracy" : 10, "velocity" : 2, "heading" : 286, "altitude" : 10, "verticalAccuracy" : 24 } ... ]}

*A JSON file is a file format that holds data like shown above.
The Rewind needs your time and location data to generate image sequence of routes that will help you remember your memory.

Don't have a file that looks like that? Don't worry. It's very simple to make one!

Step 1: To generate your own JSON file, go to www.google.com/maps/timeline?pb

Step 2: Sign in to one of your gmail accounts, and you'll be taken to a page that looks like this:




Step 3: Click on the settings gear on the bottom right corner:




Step 4: Select "Download a copy of all your data," and you'll be taken to a page like below.
After ensuring that you've selected options like shown below, click "Next":




Step 5: Choose ".zip" for file type, archive size "2GB," and any of the delivery method that works best for you.
After you hit "Create archive" button, your JSON file will be delivered to you by chosen method. Now you're all set!




Once you have your JSON file ready, feel free to close out of this window then drag-and-drop
or "choose file" your JSON file to view your memory!
